2.Explique la función de los elementos que se utilizan dentro un condicional
R/=Una instrucción condicional nos permite plantear la solución a un problema considerando los distintos casos que se pueden presentar. De esta manera, podemos utilizar un algoritmo distinto para enfrentar cada caso que pueda existir.Un algoritmo es una secuencia de instrucciones que se debe seguir para resolver un problema, en otras palabras, un paso a paso para su resolución. El método es la manera que usaremos para lograr descubrir esa secuencia instructiva a través de una clase, que es una plantilla para la creación de objetos según un modelo predefinido.
